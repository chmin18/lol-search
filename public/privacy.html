개인정보처리방침

시행일: 2026년 3월 30일

LOL 전적 검색(이하 "서비스")은 이용자의 개인정보를 중요시하며, 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 수집하고 어떻게 이용하는지 안내합니다.

1. 수집하는 정보

서비스는 이용자가 직접 입력한 정보만 사용합니다.

  • Riot ID (소환사명#태그)

서비스는 별도의 회원가입, 로그인, 쿠키 수집을 하지 않습니다.

2. 정보의 이용 목적

입력된 Riot ID는 Riot Games API를 통해 공개된 게임 전적 정보를 조회하는 용도로만 사용됩니다.

3. 정보의 저장 및 보관

서비스는 이용자의 검색 정보를 서버에 저장하지 않습니다. 모든 데이터는 Riot Games API에서 실시간으로 조회되며, 검색 완료 후 별도로 보관되지 않습니다.

4. 제3자 제공

서비스는 이용자의 정보를 제3자에게 제공하지 않습니다.

5. Riot Games API 이용

서비스는 Riot Games API를 통해 데이터를 제공받으며, Riot Games의 API 이용 정책을 준수합니다. Riot Games는 본 서비스를 보증하거나 후원하지 않습니다.
